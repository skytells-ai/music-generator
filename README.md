# AI Music Generator

Generate full songs from lyrics and style prompts using Skytells models.

## Setup

1. Create a Skytells account and get your API key from https://console.skytells.ai/settings/api-keys
2. Copy `.env.example` to `.env.local`
3. Configure your environment variables
4. Install dependencies and run the app

## Skytells Console

Use https://console.skytells.ai to manage your account, projects, and API keys.
